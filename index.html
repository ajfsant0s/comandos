Comandos de programação

Falaremos sobre os principais comandos do HTML e do CSS, sendo eles: "h1" "p" "img"

Comando "h1"

Serve para escrever títulos, reservado para o título principal

Comando "P"

Serve para escrever parágrafo, um texto mais padronizado

Comando "img"

utilizada para inserir imagem no texto da seguinte forma: "img src="nome-da-imagem.jpg"

Formatação de cor

É utilizada para mudar a cor de texto com a propiedade "color" da seguinte maneira: "color: red;" as cores devem ser escritas em inglês ou pelo "rgb(255, 0, 0);" que é #000000

Formatação de centralização

Para deixar centralizado o texto é utilizada a propiedade "text-align: center;"

Formatação de tamanho

Usada para estlizar o texto usamos a propiedade "font-size: númeropx;"
